1. What is the Internet?

The Internet is a network that connects millions of computers around the world. It allows people to share information and communicate with each other.

Examples

  • Browsing websites
  • Watching YouTube
  • Using Facebook
  • Sending emails

2. How the Internet Works

When you open a website, your browser sends a request to a server. The server sends the webpage back to your browser.

Steps

  1. Open a web browser.
  2. Type a website address.
  3. Press Enter.
  4. The server sends the webpage.
  5. The browser displays the webpage.

5. What is DNS?

DNS stands for Domain Name System. It changes a website name like www.google.com into an IP address that computers can understand.
